# 📖 大學 聲讀 Study

본 플랫폼은 한의학 및 동양 고전의 정수인 《大學》을 성현의 방식 그대로 소리 내어 읽으며 체득하는 聲讀(성독) 학습 공간입니다. 1695년 중간본 《대학언해》의 체제를 바탕으로 디지털 음성 인식 기술을 접목하여 제작되었습니다.

---

## 🎙️ 학습 안내

### 1. 성독 챌린지 (Main Mode)
* **목표:** 經 1장부터 傳 10장까지의 원문을 정확한 음으로 낭독합니다.
* **방법:** 마이크 버튼을 누르고 화면에 나타난 漢子 원문을 또박또박 읽으세요.
* **시스템:** 5문항마다 난이도가 상승하며, 타이머와 정확도 판별 시스템이 가동됩니다.
* **비고:** 음성 인식의 효율을 위해 토(吐)를 제외한 **본문 음** 위주로 판별합니다.

### 2. 난이도 구성
* **입풀기 (1~5번):** 여유로운 시간과 너그러운 판별
* **집중 (6~10번):** 표준 속도와 정확도 요구
* **정진 (11~15번):** 빠른 호흡과 높은 정교함
* **심화 (16번~):** 고도의 집중력을 요하는 마스터 단계
