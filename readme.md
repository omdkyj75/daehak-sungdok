# 🏛️ 大學章句序 & 諺解 학습 플랫폼

본 저장소는 **大學**의 핵심 문장을 시각(스와이프)과 청각(성독)을 통해 입체적으로 학습할 수 있도록 제작되었습니다.
---

## 🎮 제공 학습 모드

### 1. 漢子 訓音 스와이프 (index.html)
* **목표:** 한자의 모양을 보고 올바른 뜻과 소리를 직관적으로 판별합니다.
* **방법:** 정답은 오른쪽(O), 오답은 왼쪽(X)으로 카드를 밀어내세요.
* **기회:** 총 3번의 하트(❤️)가 주어집니다.

### 2. 大學諺解 聖讀 챌린지 (reading.html)
* **목표:** 經 1장부터 傳 10장까지의 원문을 언해본 토(吐)에 맞춰 직접 소리 내어 읽습니다.
* **방법:** 마이크 버튼을 누르고 화면의 문장을 낭독하면 AI가 정답 여부를 판별합니다.
* **특징:** 1695년 중간본 언해의 리듬을 그대로 체험할 수 있습니다.
